## 📌 Overview

This section covers setting up a **local Kubernetes cluster** using **Kind** and **kubectl**.

It focuses on installing the required tools, creating a Kubernetes cluster, configuring a multi-node cluster, verifying cluster health, and using basic cluster management commands.
