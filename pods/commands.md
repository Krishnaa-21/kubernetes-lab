# Pod Commands

## Create Pod

Create a Pod using a YAML manifest.

```bash
kubectl apply -f pod.yaml
```

---

## List Pods

View all Pods in the current namespace.

```bash
kubectl get pods
```

View Pods with more details.

```bash
kubectl get pods -o wide
```

Watch Pod status continuously.

```bash
kubectl get pods -w
```

---

## Describe Pod

Display detailed information about a Pod.

```bash
kubectl describe pod nginx-pod
```

---

## View Pod Logs

View logs generated by a Pod.

```bash
kubectl logs nginx-pod
```

Follow logs in real time.

```bash
kubectl logs -f nginx-pod
```

---

## Execute Commands Inside Pod

Open an interactive shell inside a Pod.

```bash
kubectl exec -it nginx-pod -- /bin/bash
```

For BusyBox containers:

```bash
kubectl exec -it nginx-pod -- sh
```

---

## View Pod YAML

Display the Pod configuration.

```bash
kubectl get pod nginx-pod -o yaml
```

---

## View Pod JSON

```bash
kubectl get pod nginx-pod -o json
```

---

## View Pod Labels

```bash
kubectl get pod --show-labels
```

---

## Filter Pods by Label

```bash
kubectl get pods -l app=nginx
```

---

## View Pod Events

```bash
kubectl describe pod nginx-pod
```

Check the Events section at the bottom.

---

## Delete Pod

Delete a specific Pod.

```bash
kubectl delete pod nginx-pod
```

---

## Delete Using YAML

```bash
kubectl delete -f pod.yaml
```

---

## Force Delete Pod

```bash
kubectl delete pod nginx-pod --force --grace-period=0
```

---

## Port Forward Pod

Access a Pod locally.

```bash
kubectl port-forward pod/nginx-pod 8080:80
```

Open:

```text
http://localhost:8080
```

---

## Copy Files To Pod

```bash
kubectl cp file.txt nginx-pod:/tmp/file.txt
```

---

## Copy Files From Pod

```bash
kubectl cp nginx-pod:/tmp/file.txt .
```

## Debugging Commands

Check Pod Status:

```bash
kubectl get pods
```

Inspect Pod:

```bash
kubectl describe pod nginx-pod
```

Check Logs:

```bash
kubectl logs nginx-pod
```

Execute Inside Pod:

```bash
kubectl exec -it nginx-pod -- sh
```

---

## Useful One-Liners

Count Pods:

```bash
kubectl get pods --no-headers | wc -l
```

Delete All Pods:

```bash
kubectl delete pods --all
```

Get Pod Names:

```bash
kubectl get pods -o name
```

View Running Pods Only:

```bash
kubectl get pods --field-selector=status.phase=Running
```
